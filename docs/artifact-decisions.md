# 산출물 채택/제외 결정 메모

> 이 문서는 Agent 지침이 아닌 사용자 참고용 메모입니다.

---

## 제외된 산출물

### `CLAUDE.md`
- **제외 이유**: `AGENTS.md`가 더 체계적으로 동일한 역할(AI 공통 규칙 주입)을 수행하고 있어 중복이다.
- **재도입 조건**: 이 프로젝트를 독립적인 Claude Code 세션에서 반복 실행할 경우, `AGENTS.md` 핵심 규칙을 요약한 `CLAUDE.md`로 분리하면 자동 로딩 이점을 활용할 수 있다.

### `tasks.md`
- **제외 이유**: `conversation/` 디렉토리의 Phase별 이력 문서가 더 풍부한 맥락(의사결정 근거 포함)을 제공한다. 체크박스 목록은 완성 포트폴리오에서 미완성 항목이 남을 경우 오히려 마이너스 인상을 줄 수 있다.
- **재도입 조건**: 장기 프로젝트로 확장하거나 여러 기여자가 병렬 작업할 경우 `tasks.md`로 진행 상황을 단순하게 공유하는 용도로 유효하다.
